x.send(channel, args)
Signature: (channel:string * args:obj []) -> unit
Modifiers: abstract

Send a message to the main process asynchronously via channel, you can also send arbitrary arguments. Arguments will be serialized in JSON internally and hence no (Event -> unit)s or prototype chain will be included. The main process handles it by listening for channel with ipcMain module.
x.sendSync(channel, args)
Signature: (channel:string * args:obj []) -> obj option
Modifiers: abstract

Send a message to the main process synchronously via channel, you can also send arbitrary arguments. Arguments will be serialized in JSON internally and hence no (Event -> unit)s or prototype chain will be included. The main process handles it by listening for channel with ipcMain module, and replies by setting event.returnValue. Note: Sending a synchronous message will block the whole renderer process, unless you know what you are doing you should never use it.
