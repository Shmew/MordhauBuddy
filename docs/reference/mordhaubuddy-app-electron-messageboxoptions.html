x.normalizeAccessKeys()
Signature: unit -> bool
Modifiers: abstract

Normalize the keyboard access keys across platforms. Default is false. Enabling this assumes & is used in the button labels for the placement of the keyboard shortcut access key and labels will be converted so they work correctly on each platform, & characters are removed on macOS, converted to on Linux, and left untouched on Windows. For example, a button label of Vie&w will be converted to View on Linux and View on macOS and can be selected via Alt-W on Windows and Linux.
