x.quitAndInstall(...)
Signature: (isSilent:bool * isForceRunAfter:bool) -> unit
Modifiers: abstract

Restarts the app and installs the update after it has been downloaded. It should only be called after update-downloaded has been emitted.

Note: autoUpdater.quitAndInstall() will close all application windows first and only emit before-quit event on app after that. This is different from the normal quit event sequence.

  • isSilent - windows-only Runs the installer in silent mode. Defaults to false.
  • isForceRunAfter - Run the app after finish even on silent install. Not applicable for macOS. Ignored if isSilent is set to false.
