MordhauBuddy

Snack

Namespace: MordhauBuddy.App

Fairly reusable snackbar Elmish component/library. See Snackbars.fs for example usage. Simple, conforms to Material UI, e.g. no stacking of multiple snackbars. Also no color override, but that should be easy to implement if desired.
