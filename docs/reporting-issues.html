MordhauBuddy

Reporting Issues

MordhauBuddy logs runtime data to text files that are cleaned up as they grow stale or become too numerous to keep file sizes low. Please include this data as necessary when reporting issues.

If you encounter any issues while using this application you can navigate to the following locations:

  • Windows
    • C:\Users\<user>\AppData\Local\mordhau-buddy-updater\logging
  • Linux
    • ~/.local/share/mordhau-buddy/logging
